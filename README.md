# 깡비서 급여조견표 생성기

급여테이블(변경후) 엑셀을 올리면 직무별 급여조견표 PNG를 만드는 도구입니다.

- 브라우저에서만 동작 (업로드 파일은 서버로 전송되지 않음)
- `급여테이블(변경후)` 시트 자동 인식
- 직무 선택 · 예외/특례 편집 · PNG/ZIP 다운로드

## 사용

1. 사이트 접속
2. 엑셀(`.xlsx`) 업로드
3. 직무 선택 후 조견표 생성 · 이미지 저장

## 로컬 실행

`index.html` 을 브라우저로 열면 됩니다. (인터넷 필요: 엑셀/이미지 라이브러리 CDN)
